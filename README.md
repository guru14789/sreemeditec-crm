# Nirva ERP & CRM

Nirva (Medequip CRM) is a comprehensive ERP and CRM solution tailored for Medical Equipment Enterprises. It consolidates sales, inventory, accounting, and compliance into a unified dashboard.

## Quick Links
- [Detailed Documentation](DOCUMENTATION.md)
- [Innovation Proposal](innovation_proposal.md)

## Core Modules
- **Lead CRM**: Manage sales pipeline and follow-ups.
- **Inventory**: Track medical equipment, consumables, and spares.
- **Doc Maker**: Automated PDF generation for Invoices, Quotations, and Service Reports.
- **Finance & Compliance**: Full accounting suite with GST/TDS reporting and Tally integration.
- **Employee Workspace**: Task management, attendance, and performance tracking.

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Technology Stack
- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Firebase (Firestore)
- **PDF**: jsPDF
- **AI**: Google Gemini API
